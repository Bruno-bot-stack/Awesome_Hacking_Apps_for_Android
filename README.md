## üîç Network Analysis & Traffic Interception

### **tPacketCapture**

- **Description:** Capture and analyze network traffic without root access, saving PCAP files for review.
- **Root Required:** No
- **Compatibility:** Android 4.0 and up
- **Download:** [Google Play Store](https://play.google.com/store/apps/details?id=jp.co.taosoftware.android.packetcapture)

### **Arpspoof**

- **Description:** Perform ARP spoofing to intercept and manipulate network traffic.
- **Root Required:** Yes
- **Compatibility:** Varies
- **Download:** [GitHub](https://github.com/anasm-17/Android-Arpspoof)

### **Network Discovery**

- **Description:** Scan networks for active devices and open ports to evaluate vulnerabilities.
- **Root Required:** No
- **Compatibility:** Android 1.5 and up
- **Download:** [Google Play Store](https://play.google.com/store/apps/details?id=info.lamatricexiste.network)

### **WiFiKill**

- **Description:** Disable internet connections for other devices on the same Wi-Fi network.
- **Root Required:** Yes
- **Compatibility:** Android 4.0 and up
- **Download:** [Official Site](http://wifikillapkdownload.com/)

### **NetCut**

- **Description:** Control the bandwidth usage of devices connected to your Wi-Fi network.
- **Root Required:** Yes
- **Compatibility:** Android 4.0 and up
- **Download:** [Google Play Store](https://play.google.com/store/apps/details?id=com.arcai.netcut)

### **WPS Connect**

- **Description:** Test Wi-Fi networks for WPS protocol vulnerabilities and gain access if insecure.
- **Root Required:** Yes
- **Compatibility:** Android 4.0 and up
- **Download:** [Google Play Store](https://play.google.com/store/apps/details?id=com.ngb.wpsconnect)
